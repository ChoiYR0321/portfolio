비건 입문자를 위한 LMS 플랫폼

2025.09.03 ~ 10.14 (팀 프로젝트)

📌 요약

  • Spring Boot 기반 LMS 웹 서비스 구현
  • 전반적인 UI/UX 기획 및 디자인, 화면 구성, 인터랙션 구현 담당
  • HTML, CSS, JavaScript로 프론트 UI 구축
  • OpenAI API를 활용한 레시피 추천 결과 화면 구성

🧠 기술적 포인트 (프론트엔드)

  • 전체 서비스의 UI 흐름 기획
    • : 홈 → 강의 → 진도율 → 퀴즈 → 나의 냉장고 → 나의레시피 → 게시판
  • 사용자 행동에 따른 화면 전환, 버튼 배치, 정렬 구조 직접 설계
  • 디자인 요소 : 초록 계열 컬러, 강의 카드 · 게시판 · 모달 · 입력 폼 등 재사용 가능한 컴포넌트 CSS
  • 페이지별 UI 개발
    • 공용 : 검색 필터, 카테고리 선택, 페이징 UI
    • 강의 페이지 : 강의 카드, 카테고리, 진도율, 추천 기능
    • 게시판 : 고정 게시물, 게시글 상세, 댓글 작성 모달
  • “나의 냉장고” UI 구현
    • 냉장고 내부를 연상시키는 좌측 재료 패널 + 우측 레시피 결과 화면 레이아웃
    • 재료 버튼 스타일링(태그형 버튼)
    • 클릭 시 상단에 Tag UI로 전환되는 인터랙션
    • 레시피 결과가 뜨면 나타나는 카드 애니메이션

🔍 고찰

이번 프로젝트는 사용자에게 실제로 서비스한다고 생각하며 자연스러운 UX 흐름을 만드는 것에 중점을 두었습니다. 사용자의 이용 경로, 버튼을 눌렀을 때 피드백, 성격이 다른 페이지들을 하나의 서비스처럼 통일감 있게 구성하는 방법에 대해 구체적으로 생각하고 구현하는 과정을 거쳤습니다.

또한 5명의 팀원들과 정기적인 회의를 통해 각 파트의 진행 상황을 공유하고, 프론트엔드·백엔드 간 요구사항을 조율하며 의사소통의 중요성을 체감할 수 있었습니다. 특히 구현 과정에서 발생한 UI/UX 개선 포인트나 사용성 이슈를 팀원들과 상호 피드백하며 수정해 나가면서 결과물의 서비스 품질을 끌어올릴 수 있었습니다.

이런 다양한 부분을 직접 고민하고 구현하면서 UI 설계 능력, 구조적 CSS 작성 능력, 화면 일관성 유지 능력이 크게 성장하였습니다.
