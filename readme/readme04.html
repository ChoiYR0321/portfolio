응급처치 LMS 교육 플랫폼

2025.11.03 ~ 12.01 (팀 프로젝트)

📌 요약

  • Spring Boot 기반 LMS 웹 서비스 구현
  • User 화면 UI/UX 기획 및 디자인, 화면 구성, 인터랙션 구현 담당
  • HTML, CSS, JavaScript 프론트 UI 구축
  • 전체적으로 학습 목적과 흐름을 직관적으로 전달하는 화면 UI 설계

🧠 기술적 포인트 (프론트엔드)

  • 교육용 LMS에 맞는 전체 UI 흐름 설계
    • : 강의 목록 → 수강 신청 → 수강 목록 → 강의 시청 → 진도 관리 → 시험 → 교육용 게임 → 수료 / 미수료 관리 → 이수증 발급
  • 서비스 첫 진입을 고려한 인트로 화면 UI 설계
    • LMS 서비스 목적과 학습 흐름을 한눈에 전달하는 인트로 구성
    • 핵심 기능 위주의 시각적 안내 방식 적용
    • 버튼 배치와 정보 노출 우선순위를 통해 자연스러운 다음 행동 유도
    • 도트 네비게이션과 스크롤 기반 인터랙션을 활용한 인트로 화면 UX 설계
    • 실제 서비스 메인 페이지를 고려한 구조로 설계하여 이후 화면과의 연결성 유지
  • 강의 목록 페이지 UI 구성
    • 상단 탭 기반 카테고리(전체 / 구조자 / 자가 응급처치) 전환 UI
    • 카드형 강의 리스트(썸네일 · 카테고리 태그 · 요약 · 수강 신청 버튼)
  • 강의 영상 재생 팝업 UI
    • 모달 형태의 강의 재생 팝업 (좌: 영상 / 우: 목차·메모)
    • 탭 전환 UI를 통한 ‘강의 목차 / 메모’ 분리
    • 차시별 선택 상태(active), 잠금 상태(locked) 시각적 구분
    • 영상 시청 흐름에 맞춘 버튼 배치 및 완료 UX 설계
  • 게임형 학습 요소를 반영한 배지 보관함 UI 구현
    • 응급처치 학습 완료 결과를 시각적으로 확인할 수 있는 배지 수집 구조 설계
    • 배지 획득 / 미획득 상태에 따른 locked · unlocked UI
    • 진행도 카운트 및 프로그레스 바로 학습 성취도 가시화
    • 배지 카드 hover 시 설명 툴팁 노출로 정보 전달 최소화
    • 키트 이미지 위에 배지를 배치하여 게임 맵처럼 인식되는 레이아웃 구성
    • 학습 동기를 유도하는 시각적 보상 구조 설계

🔍 고찰

이번 프로젝트에서는 개발에 앞서 기획 문서와 화면 흐름을 먼저 정리하고, 전체 일정과 우선순위를 기준으로 작업을 진행하였습니다. 그 덕분에 기능 구현에만 집중하기보다, “이 서비스가 실제로 사용된다면 어떤 흐름으로 이용될까”를 계속 염두에 두고 UI를 설계할 수 있었습니다.

특히 LMS라는 서비스 특성에 대해 이전보다 더 깊이 고민하게 되었습니다. 강의 탐색, 학습 진행, 상태 확인이 반복적으로 이루어지는 구조이기 때문에 사용자가 현재 위치를 쉽게 인지하고, 다음 행동을 자연스럽게 선택할 수 있도록 하는 것이 중요하다고 느꼈습니다. 이에 따라 사이드바, 모달 화면 등 학습 흐름을 보조하는 화면 구성에 많은 신경을 썼습니다.

프론트엔드 코드 구조 또한 이전보다 더 세분화하여 정리했습니다. 공통 컴포넌트, 페이지별 스크립트를 분리하면서 기능 추가나 수정 시 영향을 최소화하려고 노력했고, 코드를 ‘일단 동작하게 만드는 것’보다 ‘유지·확장 가능한 구조’로 작성하는 중요성을 체감할 수 있었습니다.

구현 과정에서 가장 어려웠던 부분은 드롭다운 구조를 가진 사이드바였습니다. 단순히 열고 닫는 기능을 넘어서, 페이지 이동 후에도 사이드바가 펼쳐진 상태로 유지되어야 할지, 혹은 접힌 상태가 더 자연스러운지 판단하는 과정 자체가 쉽지 않았습니다. 또한 active 상태와 서브메뉴 노출을 함께 관리하다 보니 이벤트 처리와 상태 제어가 복잡해져 여러 차례 코드가 꼬이는 문제를 겪었습니다.

이러한 문제는 사용자 경험과 서비스 성격을 기준으로 판단해야 하는 영역이라 생각하여, 여러 방식으로 시도하고 수정하는 과정을 반복하며 점차 안정적인 구조로 정리할 수 있었습니다. 이 경험을 통해 UI 구현에서도 ‘어떻게 보이느냐’뿐 아니라 ‘어떤 상태를 유지해야 하는지’를 함께 고려하는 시각을 갖게 되었습니다.
