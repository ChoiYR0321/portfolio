2025.09.03 ~ 10.14 (팀 프로젝트)

1. 로그인 전 메인화면

  • 상단 헤더 메뉴바 생성

3. 로그인

  • 강의 카드 : 카테고리, 진도율, 재생 버튼, 퀴즈 풀기 버튼, 추천 기능

5. 강의 재생 모달

  • 강의 제목, 강의 영상, 강의 설명, 수강 완료 버튼

  • 오답 문제 정리

6. 나의 냉장고 초기 화면

  • 오른쪽 화면: 검색 기능, 재료 카테고리 태그
  • 왼쪽 화면: 레시피 찾기 버튼

7. 나의 냉장고 레시피 카드 생성

  • 레시피 카드: 맘에드는 레시피 저장 버튼으로 저장 가능
  • 저장 된 레시피는 "마이페이지 -> 나의 레시피"에서 확인 가능

8. 나의 냉장고 레시피 카드 생성

  • 비건 재료 외의 재료 선택 시 문구와 함께 해당 재료를 제외한 레시피 추천

  • 고정 게시물 표시
  • 분류 / 제목 / 작성자 / 작성일

10. 게시물 상세

11. 게시물 상세 (공지 사항)
